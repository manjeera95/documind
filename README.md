# DocuMind

Enterprise AI Document Intelligence Platform built with FastAPI, PostgreSQL, OpenAI, and pgvector.

## Overview

DocuMind is a backend platform that enables users to upload documents, extract meaningful information, perform semantic search, and interact with documents using Retrieval-Augmented Generation (RAG).

The platform is designed using modern backend engineering practices with a focus on scalability, security, and maintainability.

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- pgvector
- SQLAlchemy
- Alembic
- OpenAI
- LangChain
- Docker
- JWT Authentication
- GitHub Actions
